# Windows 文件筛选与移动工具 - 更新说明

## 版本 2.0.0 更新内容

根据用户需求，我们对工具进行了重大更新，主要包含以下两个核心改进：

### 🎯 主要更新

#### 1. 压缩包支持
- **原功能**: 选择文件夹进行搜索
- **新功能**: 选择压缩包自动解压并搜索
- **支持格式**: ZIP、RAR、7Z
- **自动化**: 自动解压、搜索、清理临时文件

#### 2. 关键字输入方式改进
- **原方式**: 单行输入，逗号分隔
- **新方式**: 多行文本框，每行一个关键字
- **优势**: 更直观、支持更多关键字、避免逗号冲突

### 🔧 技术实现

#### 新增依赖
```bash
pip install rarfile py7zr
```

#### 新增模块
- `zipfile`: Python标准库，处理ZIP文件
- `rarfile`: 第三方库，处理RAR文件
- `py7zr`: 第三方库，处理7Z文件
- `tempfile`: Python标准库，管理临时文件

#### 核心新功能函数
1. `extract_archive()`: 解压压缩包到临时目录
2. `validate_archive()`: 验证压缩包格式和可访问性
3. `find_and_move_files_from_archive()`: 从压缩包中查找并移动文件
4. `count_matching_files_in_archive()`: 统计压缩包中匹配的文件数量
5. `cleanup_temp_directory()`: 清理临时目录

### 🎨 界面更新

#### 文件选择区域
- **标题**: "选择压缩包 (支持 .zip, .rar, .7z)"
- **文件类型过滤**: 自动过滤支持的压缩包格式
- **路径显示**: 显示选中的压缩包完整路径

#### 关键字输入区域
- **多行文本框**: 替代原来的单行输入框
- **滚动条**: 支持大量关键字输入
- **提示文本**: "输入关键字 (每行一个关键字)"
- **快捷键**: Ctrl+Enter 快速开始处理

#### 功能增强
- **预览功能**: 支持压缩包内文件预览
- **多线程处理**: 解压和预览都在后台线程执行
- **进度指示**: 更详细的处理状态显示
- **自动清理**: 程序关闭时自动清理临时文件

### 🚀 使用示例

#### 压缩包处理示例
```
1. 选择压缩包：project.zip
2. 输入关键字：
   config
   settings

3. 点击预览查看匹配文件数
4. 确认后开始处理
5. 文件自动移动到桌面专用文件夹
```

### ⚠️ 注意事项

#### 重要提醒
1. **文件移动**: 程序执行的是移动操作，不是复制
2. **临时空间**: 解压需要足够的临时磁盘空间
3. **RAR支持**: RAR文件需要系统安装WinRAR或UnRAR
4. **权限要求**: 需要压缩包读取权限和桌面写入权限

#### 建议操作
1. 重要文件请提前备份
2. 先使用预览功能确认文件数量
3. 确保有足够的磁盘空间
4. 关闭程序前确认操作完成

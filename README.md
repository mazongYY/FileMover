# Windows 文件筛选与移动工具 - 支持压缩包

一个功能强大的Windows文件管理工具，支持从压缩包中筛选文件并移动到桌面。

## 功能特性

- 📦 **压缩包支持**: 支持ZIP、RAR、7Z格式的压缩包自动解压
- 🔍 **智能搜索**: 递归搜索压缩包内所有文件和子目录
- 🏷️ **关键字筛选**: 支持按行输入多个关键字，不区分大小写
- 📁 **智能分类**: 自动将文件分类到"命中文件"和"未命中文件"文件夹
- 📂 **项目管理**: 文件保存在项目根目录下，便于管理
- 🛡️ **安全保护**: 自动处理重名文件，避免覆盖
- 📊 **预览功能**: 在移动前预览匹配和未匹配的文件数量
- 📝 **完整日志**: 详细的日志记录系统，支持文件和控制台输出
- 🎯 **用户友好**: 直观的图形界面，操作简单
- 🧹 **自动清理**: 自动清理临时解压文件
- 🔧 **自动初始化**: 程序启动时自动检测和创建必要目录

## 系统要求

- Windows 7/8/10/11
- Python 3.6 或更高版本
- 必需依赖：rarfile, py7zr（用于压缩包支持）
- 可选：WinRAR或UnRAR程序（用于RAR文件支持）

## 安装和使用

### 方法一：直接运行Python脚本

1. 确保已安装Python 3.6+
2. 安装必需的依赖包：
   ```bash
   pip install rarfile py7zr
   ```
3. 下载项目文件到本地
4. 双击运行 `main.py` 或在命令行中执行：
   ```bash
   python main.py
   ```

### 方法二：打包为可执行文件

1. 安装PyInstaller：
   ```bash
   pip install pyinstaller
   ```

2. 打包程序：
   ```bash
   pyinstaller --onefile --windowed main.py
   ```

3. 运行生成的 `dist/main.exe`

## 使用说明

1. **选择压缩包**: 点击"浏览"按钮选择要处理的压缩包（支持.zip、.rar、.7z格式）
2. **输入关键字**: 在多行文本框中输入关键字，每行一个关键字
3. **预览（可选）**: 点击"预览匹配文件数"查看将要移动的文件数量
4. **开始处理**: 点击"开始处理"按钮执行解压和文件移动操作
5. **查看结果**: 在操作日志中查看详细的处理结果

### 快捷键
- **Ctrl+Enter**: 在关键字输入框中按此组合键可快速开始处理

## 注意事项

- ⚠️ **权限要求**: 确保对压缩包有读取权限，对桌面有写入权限
- 🔄 **文件移动**: 程序会移动（而非复制）文件，请谨慎操作
- 📂 **目标位置**: 文件会移动到桌面的"筛选文件_时间戳"文件夹中
- 🔍 **搜索范围**: 程序会递归搜索压缩包内所有子目录
- 📝 **重名处理**: 如遇重名文件，会自动添加数字后缀
- 🧹 **临时文件**: 程序会自动清理解压产生的临时文件
- 📦 **压缩包格式**: 支持ZIP、RAR、7Z格式，RAR需要系统安装WinRAR或UnRAR

## 技术实现

- **GUI框架**: Tkinter (Python标准库)
- **文件操作**: os, shutil (Python标准库)
- **压缩包处理**: zipfile (标准库), rarfile, py7zr
- **多线程**: threading (避免界面卡顿)
- **临时文件**: tempfile (Python标准库)
- **日志系统**: logging, RotatingFileHandler (Python标准库)
- **类型提示**: typing (提高代码可读性)

# Windows 文件筛选与移动工具

本项目旨在开发一个简单的 Windows 应用程序，允许用户通过图形界面选择文件夹，并输入关键字来筛选该文件夹中的文件。随后，应用程序会将匹配关键字的文件移动到用户的桌面。

该应用程序使用 Python 编写，结合 Tkinter 进行 GUI 开发，并利用标准库进行文件操作。适用于 Windows 系统环境。

### # Windows 文件筛选与移动工具 v3.0 更新说明

#### 🎯 版本 3.0.0 重大更新

根据用户需求，我们对工具进行了三个核心功能的重大改进：

### 🆕 主要新功能

#### 1. 完整的日志系统

- **日志文件**: 自动创建 `logs/file_filter.log` 文件
- **轮转日志**: 支持日志文件轮转，单文件最大10MB，保留5个备份
- **多级别日志**: 支持DEBUG、INFO、WARNING、ERROR四个级别
- **双重输出**: 同时输出到文件和控制台
- **GUI集成**: 在界面中实时显示日志信息
- **时间戳**: 每条日志都包含详细的时间戳

#### 2. 项目根目录文件管理

- **本地存储**: 解压文件存放在项目根目录的 `extracted_files` 文件夹
- **自动检测**: 程序启动时自动检测目录是否存在
- **自动创建**: 如果目录不存在，自动创建完整的目录结构
- **目录结构**:

  ```
  extracted_files/
  ├── 命中文件/      # 匹配关键字的文件
  └── 未命中文件/    # 未匹配关键字的文件
  ```

#### 3. 智能文件分类系统

- **全文件处理**: 处理压缩包中的所有文件，不遗漏任何文件
- **智能分类**: 根据关键字匹配结果自动分类
- **命中文件**: 文件名包含关键字的文件移动到"命中文件"文件夹
- **未命中文件**: 文件名不包含关键字的文件移动到"未命中文件"文件夹
- **统计信息**: 详细显示命中和未命中的文件数量

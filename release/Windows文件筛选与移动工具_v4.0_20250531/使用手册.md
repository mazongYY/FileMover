# Windows 文件筛选与移动工具 - 使用手册

## 🎯 快速开始

### 基本操作流程
1. **选择压缩包**：点击"选择压缩包"按钮或直接拖拽压缩包到程序界面
2. **输入关键字**：在关键字输入框中输入要搜索的关键字（每行一个）
3. **预览结果**：点击"预览匹配文件"查看将要处理的文件
4. **开始处理**：点击"开始处理"执行文件筛选和移动操作

### 控制面板功能

#### 1. 压缩包选择
- **选择压缩包**：浏览并选择要处理的压缩包文件
- **支持格式**：ZIP、RAR、7Z等主流格式
- **拖拽支持**：可直接拖拽压缩包到程序界面

#### 2. 关键字输入
- **输入方式**：每行输入一个关键字
- **匹配模式**：默认为包含匹配，支持正则表达式
- **示例**：
  ```
  报告
  文档
  2024
  ```

#### 3. 高级设置
- **正则表达式**：启用复杂的模式匹配
- **文件类型过滤**：按文件扩展名筛选
- **操作模式**：选择移动、复制或创建链接
- **文件大小过滤**：按文件大小范围筛选
- **日期过滤**：按文件修改时间筛选

## 🔧 高级功能详解

### 1. 正则表达式匹配
启用正则表达式后，可以使用复杂的匹配模式：

**常用正则表达式示例**：
```
.*\.pdf$          # 匹配所有PDF文件
^报告.*2024       # 匹配以"报告"开头且包含"2024"的文件
\d{4}-\d{2}-\d{2} # 匹配日期格式 YYYY-MM-DD
```

### 2. 文件类型过滤
可以按以下类型筛选文件：
- **文档**：.doc, .docx, .pdf, .txt, .rtf
- **图片**：.jpg, .png, .gif, .bmp, .tiff
- **视频**：.mp4, .avi, .mkv, .mov, .wmv
- **音频**：.mp3, .wav, .flac, .aac
- **压缩包**：.zip, .rar, .7z, .tar
- **自定义**：手动输入文件扩展名

### 3. 操作模式
- **移动模式**：将文件从压缩包移动到目标目录
- **复制模式**：复制文件到目标目录，保留原始压缩包
- **链接模式**：创建文件的硬链接（节省空间）

### 4. 撤销功能
- **自动记录**：每次操作都会自动记录
- **一键撤销**：可以撤销最近的操作
- **批量撤销**：支持撤销多个操作
- **历史查看**：查看详细的操作历史

## 🔍 故障排除

### 常见问题解决

**1. 压缩包无法打开**
- 检查文件是否损坏
- 确认支持的格式（ZIP/RAR/7Z）
- 对于RAR文件，确保系统已安装WinRAR

**2. 找不到匹配文件**
- 检查关键字拼写
- 尝试使用部分关键字
- 启用正则表达式模式

**3. 程序运行缓慢**
- 减少同时处理的文件数量
- 关闭不必要的文件类型过滤
- 确保有足够的磁盘空间

**4. 撤销操作失败**
- 检查目标文件是否被其他程序占用
- 确认有足够的权限
- 查看日志文件了解详细错误

## 📊 日志和监控

### 日志文件位置
- **路径**：`logs/file_filter_YYYYMMDD.log`
- **内容**：详细的操作记录和错误信息
- **格式**：时间戳 + 日志级别 + 消息内容

### 监控信息
- **处理进度**：实时显示当前处理状态
- **文件统计**：显示匹配和未匹配文件数量
- **操作历史**：记录所有操作的详细信息

Windows 文件筛选与移动工具 v3.0 - 智能分类版 - 使用指南
========================================================

📋 快速开始
-----------
1. 安装依赖：pip install rarfile py7zr
2. 双击 run.bat 启动程序（需要Python环境）
   或
3. 双击 main.py 直接运行（需要Python环境）
   或
4. 使用打包后的exe文件（无需Python环境）

🔧 打包为EXE
------------
1. 双击 build.bat 自动打包
2. 等待打包完成
3. 在 dist 文件夹中找到生成的exe文件

📖 使用步骤
-----------
1. 点击"浏览"按钮选择要处理的压缩包（支持.zip、.rar、.7z）
2. 在多行文本框中输入关键字（每行一个关键字）
3. 可选：点击"预览匹配文件数"查看命中/未命中文件统计
4. 点击"开始处理"执行解压和文件分类
5. 查看操作日志了解详细处理结果
6. 在项目根目录的extracted_files文件夹中查看分类结果

⌨️ 快捷键
---------
• Ctrl+Enter: 在关键字输入框中快速开始处理

💡 使用技巧
-----------
• 关键字不区分大小写
• 支持中文关键字
• 每行输入一个关键字，支持多行输入
• 程序会自动解压并递归搜索所有子文件夹
• 重名文件会自动重命名，不会覆盖
• 所有文件会智能分类到"命中文件"和"未命中文件"文件夹
• 程序会自动清理临时解压文件
• 详细操作日志保存在logs/file_filter.log文件中

🆕 v3.0新功能
-----------
• 📝 完整日志系统：所有操作都有详细记录
• 📂 项目目录管理：文件保存在项目根目录下
• 🎯 智能文件分类：自动分类所有文件
• 🔧 自动目录检测：程序启动时自动创建必要目录
• 📊 详细统计信息：显示命中和未命中文件数量

⚠️ 注意事项
-----------
• 程序会移动（不是复制）文件，请谨慎操作
• 建议先使用"预览"功能确认文件数量
• 确保对压缩包有读取权限，对桌面有写入权限
• 重要文件请提前备份
• RAR文件需要系统安装WinRAR或UnRAR程序

🔍 使用示例
-----------
示例1 - 查找图片文件：
  压缩包：photos.zip
  关键字（每行一个）：
    jpg
    png
    gif
    jpeg
    bmp

示例2 - 查找文档文件：
  压缩包：documents.rar
  关键字（每行一个）：
    doc
    pdf
    txt
    ppt
    报告

示例3 - 查找特定项目文件：
  压缩包：project.7z
  关键字（每行一个）：
    config
    设置
    配置

示例4 - 查找备份文件：
  压缩包：backup.zip
  关键字（每行一个）：
    backup
    备份
    bak

🛠️ 系统要求
-----------
• Windows 7/8/10/11
• Python 3.6+ (如果运行源码)
• 必需依赖：rarfile, py7zr
• 可选：WinRAR或UnRAR程序（用于RAR文件）

📞 技术支持
-----------
如遇问题，请检查：
1. Python版本是否为3.6或更高
2. 是否有足够的磁盘空间
3. 是否有相应的文件夹访问权限
4. 关键字格式是否正确
